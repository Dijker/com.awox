AwoX ist Spezialist für vernetzte Beleuchtung.
Das Unternehmen verfügt über eine sehr breite Palette an vernetzten Beleuchtungsprodukten.
Diese werden auch unter der Marke EGLO Connect by AwoX vertrieben

Passen Sie die RGB-Farbe und Helligkeit an, stellen Sie Ihre bevorzugte Weißtemperatur ein und schöpfen Sie das volle Potenzial Ihrer Geräte über die AwoX-App auf Homey aus

Die App ist eine Community initiative.
Um zusätzliche Geräte anzufordern, verwenden Sie bitte das Athom Homey Community Forum.

Momentan wird für die Geräte nur Zigbee unterstützt, kein Bluetooth.

Anmerkung:
Bis Firmware 2.3.9 verwendete AwoX eine gemeinsame Zigbee-Produkt-ID für mehrere Gerätetypen (Dim, Tunable, RGB, E27, GU10, LED-Panels usw.)
Ab Firmware 2.3.12 gibt AwoX verschiedenen Produkten eine eigene Zigbee ProductID.
Hinweis: Aktualisieren Sie zuerst die Firmware über die Mobile AwoX App per Bluetooth, sonst führt dies später zu Problemen.
Siehe Community-Forum für die hinzugefügten Produkt-IDs
